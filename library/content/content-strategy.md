# Content Strategy Fundamentals

> **Purpose**: Plan, create, and manage content that drives business results
> **Impact**: Content is the substance of every page — get it right
> **Principle**: Create for users first, optimize for search second

---

## Content Strategy Framework

### The Content Strategy Quad

```
           PURPOSE
              │
              │
    VOICE ────┼──── AUDIENCE
              │
              │
           PROCESS
```

| Pillar | Questions to Answer |
|--------|---------------------|
| **Purpose** | Why are we creating content? What business goals? |
| **Audience** | Who are we writing for? What do they need? |
| **Voice** | How should we sound? What's our personality? |
| **Process** | How do we create, approve, publish, maintain? |

## Content Auditing

### When to Audit

- Before website redesign
- When traffic/conversions decline
- Annually for maintenance
- When entering new markets

### Content Inventory Template

| URL | Title | Type | Word Count | Last Updated | Traffic | Conversions | Action |
|-----|-------|------|------------|--------------|---------|-------------|--------|
| /page-1/ | Title | Blog | 1,500 | 2023-01-15 | 500/mo | 10 | Keep |
| /page-2/ | Title | Product | 800 | 2021-06-01 | 50/mo | 0 | Rewrite |
| /page-3/ | Title | Blog | 400 | 2019-03-20 | 5/mo | 0 | Delete |

### Audit Actions

| Action | Criteria |
|--------|----------|
| **Keep** | High traffic, converting, up-to-date |
| **Update** | Good topic, outdated info/data |
| **Rewrite** | Good topic, poor performance, can improve |
| **Consolidate** | Similar topics competing with each other |
| **Delete** | No traffic, no value, no potential |
| **Redirect** | Delete but has backlinks/authority |

## Content Hierarchy (Message Architecture)

### Priority Messaging Matrix

```
Level 1: Primary Message
├── What is this? (single sentence)
├── Why should I care? (value prop)
└── What should I do? (CTA)

Level 2: Supporting Messages
├── Key benefit 1
├── Key benefit 2
└── Key benefit 3

Level 3: Proof Points
├── Statistics
├── Testimonials
└── Case studies

Level 4: Details
├── Features
├── Specifications
└── FAQs
```

### Content Prioritization

| Priority | Content | Page Placement |
|----------|---------|----------------|
| 1 | Headline + value prop | Above fold |
| 2 | Primary CTA | Above fold |
| 3 | Key benefits (3) | Just below hero |
| 4 | Social proof | After benefits |
| 5 | Detailed features | Mid-page |
| 6 | Testimonials/Cases | Pre-CTA |
| 7 | Final CTA | Bottom |

## Content Formats & Length

### Optimal Content Length by Type

| Content Type | Optimal Length | Notes |
|--------------|----------------|-------|
| Homepage | 500-1,000 words | Scannable, focused |
| Product page | 1,000-2,000 words | Detailed but concise |
| Service page | 1,500-2,500 words | Explain value thoroughly |
| Blog post (standard) | 1,500-2,500 words | SEO sweet spot |
| Long-form guide | 3,000-7,000 words | Pillar content |
| Landing page | 500-1,500 words | Depends on offer complexity |
| Email | 50-125 words | Get to point fast |

> "Long-form content (3,000+ words) gets 3x more traffic, 4x more shares, and 3.5x more backlinks than shorter content."
>
> *Source: [Backlinko Content Study](https://backlinko.com/content-study)*

### Scannable Content Patterns

```
✓ Short paragraphs (2-4 sentences)
✓ Subheadings every 200-300 words
✓ Bullet points for lists
✓ Bold for key points
✓ Pull quotes for emphasis
✓ Images to break up text
✓ White space around elements
```

## Voice & Tone Guidelines

### Voice vs Tone

| Voice | Tone |
|-------|------|
| Consistent personality | Adapts to situation |
| Always the same | Changes per context |
| Who we are | How we speak now |
| Example: Friendly | Example: Supportive (error) vs Celebratory (success) |

### Voice Attribute Framework

Choose 3-5 attributes that define your brand voice:

| Attribute | We Are | We're Not |
|-----------|--------|-----------|
| Friendly | Approachable, warm | Cold, corporate |
| Expert | Knowledgeable, helpful | Condescending |
| Direct | Clear, concise | Wordy, vague |
| Confident | Assured, trustworthy | Arrogant, boastful |
| Human | Conversational, real | Robotic, formal |

### Tone Adaptation

| Context | Tone | Example |
|---------|------|---------|
| Marketing | Enthusiastic | "Unlock your potential!" |
| Support | Empathetic | "We understand this is frustrating." |
| Error message | Helpful | "Something went wrong. Here's how to fix it." |
| Success | Celebratory | "You did it! Welcome aboard." |
| Legal | Formal | "These terms govern your use of..." |

```
     └──────────────────────────────────────────────────────────────────────────┘
                                    (Continuous cycle)
```

## Content Briefs

### Brief Template

```markdown
# Content Brief: [Title]

## Overview
- **Target URL**: /path/to/page/
- **Target keyword**: primary keyword
- **Search intent**: informational/transactional/navigational
- **Word count**: X,XXX
- **Due date**: YYYY-MM-DD

## Audience
- **Who**: Primary persona
- **Pain points**: What problem are they solving?
- **Stage**: Awareness/Consideration/Decision

## Goals
- **Primary goal**: Drive signups
- **Secondary goal**: Rank for [keyword]

## Outline
1. Introduction (hook + value prop)
2. Section 1: [Topic]
3. Section 2: [Topic]
4. Section 3: [Topic]
5. Conclusion + CTA

## Competitor Analysis
- [Competitor 1 URL] - Notes
- [Competitor 2 URL] - Notes
- [Competitor 3 URL] - Notes

## Requirements
- [ ] Include 2-3 internal links
- [ ] Include 1-2 external sources
- [ ] Add meta title and description
- [ ] Include schema markup recommendations
```

## Measuring Content Success

### Key Metrics by Goal

| Goal | Metrics |
|------|---------|
| **Awareness** | Traffic, impressions, reach |
| **Engagement** | Time on page, scroll depth, shares |
| **Lead generation** | Conversions, form fills, downloads |
| **Sales** | Assisted conversions, revenue attribution |
| **SEO** | Rankings, backlinks, organic traffic |
| **Retention** | Return visits, email opens |

### Content Performance Tiers

| Tier | Criteria | Action |
|------|----------|--------|
| **Star** | Top 10% traffic, high conversion | Promote more, create similar |
| **Performer** | Above average | Update and optimize |
| **Average** | Middle 50% | Test improvements |
| **Underperformer** | Below average, some potential | Rewrite or consolidate |
| **Prune** | Bottom 10%, no potential | Delete or redirect |

## Content Repurposing

### One Piece, Many Formats

```
Long-form Guide (Pillar)
        │
        ├── Blog posts (sections)
        ├── Email series (key points)
        ├── Social posts (quotes, tips)
        ├── Infographic (data visualization)
        ├── Video (summary or walkthrough)
        ├── Podcast episode (discussion)
        ├── Slide deck (presentation)
        └── Webinar (deep dive)
```

### Repurposing Strategy

| Original | Repurpose Into | Effort |
|----------|----------------|--------|
| Blog post | Social threads | Low |
| Blog post | Email newsletter | Low |
| Webinar | Blog post | Medium |
| Webinar | Short video clips | Medium |
| Case study | Testimonial quotes | Low |
| Research report | Infographic | Medium |
| Guide | Checklist/template | Low |

## Content Maintenance

### Update Schedule

| Content Type | Review Frequency | Signs to Update |
|--------------|------------------|-----------------|
| Product pages | Quarterly | Features change |
| Blog posts | Annually | Outdated data, broken links |
| Guides | Semi-annually | Industry changes |
| Case studies | As needed | New results, company changes |
| Legal pages | Annually | Regulation changes |

### Content Freshness Signals

| Signal | Action |
|--------|--------|
| "Updated: [Date]" | Show recent updates |
| "Originally published: [Date]" | Historical context |
| Update logs | Show revision history |
| "Last verified: [Date]" | For data/stats |
