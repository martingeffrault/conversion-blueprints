# Competitive Analysis Guide

> **Purpose**: Systematic competitor research for CRO insights
> **Impact**: Identify optimization opportunities others have validated
> **Principle**: Learn from competitors' successes and failures—don't optimize in the dark

---

## Why Competitive Analysis Matters

### The Business Case

> "Performing competitive analysis is crucial for any successful conversion rate optimization program. Without analyzing what your competitors are doing, you are essentially optimizing your website in the dark."
>
> *Source: [Invesp](https://www.invespcro.com/blog/competitive-analysis-for-conversion-rate-optimization/)*

### Key Benefits

| Benefit | Description |
|---------|-------------|
| **Identify strengths/weaknesses** | Objectively compare your site to competitors |
| **Inspire new ideas** | Discover tests you may have never considered |
| **Justify changes** | Evidence-based proposals for stakeholders |
| **Track trends** | Stay current with industry standards |
| **Benchmark performance** | Know where you stand vs market |

## The Competitive Analysis Framework

### Phase 1: Identify Competitors

#### Types of Competitors

| Type | Definition | Example |
|------|------------|---------|
| **Direct** | Same product, same market | Nike vs Adidas |
| **Indirect** | Different product, same need | Gym vs Peloton |
| **Aspirational** | Best-in-class, any industry | Apple, Amazon |
| **SERP** | Ranking for your keywords | Organic competitors |

#### How to Find Competitors

1. **Search competitors**: Google your main keywords
2. **Industry reports**: Gartner, Forrester, G2
3. **Social listening**: Who do customers mention?
4. **Customer interviews**: Who else did they consider?
5. **Similar sites**: SimilarWeb, Alexa alternatives

### Phase 2: Define Analysis Scope

#### What to Analyze

```
Homepage
├── Value proposition clarity
├── Hero section effectiveness
├── Trust signals present
├── Primary CTA strength
└── Above-fold content

Key Landing Pages
├── Message match with ads
├── Form design
├── Social proof placement
├── Urgency elements
└── Mobile experience

Conversion Funnel
├── Steps to purchase/signup
├── Friction points
├── Cart/checkout design
├── Form length
└── Payment options

Content Strategy
├── Blog topics
├── Resource library
├── Case studies
├── Documentation
└── SEO focus areas
```

### Phase 3: Data Collection

#### Tools for Analysis

| Tool | Purpose | Free? |
|------|---------|-------|
| **SimilarWeb** | Traffic, sources, engagement | Freemium |
| **SEMrush** | Keywords, ads, backlinks | Paid |
| **Ahrefs** | SEO, content gaps | Paid |
| **BuiltWith** | Technology stack | Freemium |
| **Wappalyzer** | Tech detection | Free |
| **PageSpeed Insights** | Performance | Free |
| **Wayback Machine** | Historical changes | Free |

#### Metrics to Capture

| Category | Metrics |
|----------|---------|
| **Traffic** | Monthly visits, traffic sources, geography |
| **Engagement** | Bounce rate, pages/visit, time on site |
| **SEO** | Domain authority, ranking keywords, backlinks |
| **Performance** | Load time, Core Web Vitals |
| **Social** | Followers, engagement rate, posting frequency |
| **Ads** | Ad copy, landing pages, keywords bid |

## CRO-Specific Analysis

### Homepage Analysis Checklist

| Element | Questions to Answer |
|---------|---------------------|
| **Value Proposition** | How do they communicate unique value in <5 seconds? |
| **Hero Section** | Image vs video? Split vs centered? CTA placement? |
| **Trust Signals** | Logos, testimonials, badges above fold? |
| **Navigation** | How many items? Mega menu? CTA in nav? |
| **Social Proof** | Customer count, reviews, case studies visible? |
| **CTA Hierarchy** | Primary vs secondary actions clear? |

### Landing Page Analysis

| Element | Evaluation Criteria |
|---------|---------------------|
| **Headline** | Benefit-focused? Specific? Matches ad? |
| **Subheadline** | Expands on headline? Addresses objections? |
| **Visual** | Product image? Demo? Person using product? |
| **Form** | Fields count? Above fold? Progressive? |
| **CTA** | Action verb? Benefit-oriented? Color contrast? |
| **Proof** | Testimonials? Logos? Statistics? |
| **Urgency** | Limited time? Scarcity? Social proof? |

### Conversion Funnel Mapping

```
COMPETITOR FUNNEL MAP

Homepage → Product Page → Cart → Checkout → Confirmation

Steps: [Count for each competitor]
Forms: [Fields required at each step]
Trust: [Security badges, guarantees at checkout]

```

### Pricing Page Analysis

| Element | Questions |
|---------|-----------|
| **Tiers** | How many? What's the anchor? |
| **Naming** | Descriptive or persona-based? |
| **Features** | How presented? Checkmarks? Limits? |
| **Highlighted** | Which tier pushed? Why? |
| **CTA** | Free trial? Demo? Buy now? |
| **FAQ** | What objections addressed? |
| **Comparison** | vs competitors shown? |

## Prioritization Frameworks

### ICE Scoring (for Test Ideas from Competitors)

| Idea | Impact (1-10) | Confidence (1-10) | Ease (1-10) | ICE Score |
|------|---------------|-------------------|-------------|-----------|
| Add trust badges | 7 | 8 | 9 | 24 |
| Simplify checkout | 9 | 7 | 4 | 20 |
| Video testimonials | 6 | 5 | 6 | 17 |
| Exit intent popup | 5 | 6 | 8 | 19 |

> "In 2026, CRO leaders are leaning on frameworks like ICE (Impact, Confidence, Ease) or PIE (Potential, Importance, Ease) to prioritize tests that matter most."
>
> *Source: [WebFX](https://www.webfx.com/blog/conversion-rate-optimization/cro-trends/)*

### PIE Scoring Alternative

| Idea from Comp B | 6 | 9 | 8 | 7.7 |

## Analysis Process

### Step-by-Step Workflow

```
Week 1: Setup
├── Define competitor list (5-10)
├── Set up tracking tools
├── Create analysis templates
└── Assign team responsibilities

Week 2: Data Collection
├── Capture homepage screenshots
├── Map conversion funnels
├── Document pricing/features
├── Run speed tests
└── Analyze traffic sources

Week 3: Deep Dive
├── Sign up for competitor products
├── Go through their onboarding
├── Test support responsiveness
├── Document email sequences
└── Analyze mobile experience

Week 4: Synthesis
├── Compile findings
├── Identify patterns
├── Score opportunities
├── Create recommendations
└── Present to stakeholders
```

### What to Document

For each competitor, capture:

1. **Screenshots** - Homepage, key pages, checkout
2. **Recordings** - Screen record signup/purchase flow
3. **Emails** - Sign up to capture sequences
4. **Ads** - Document ad copy and landing pages
5. **Changes** - Track updates over time (Wayback)

## Turning Insights into Action

### From Analysis to Hypothesis

| Competitor Insight | Your Current State | Hypothesis |
|--------------------|-------------------|------------|
| Comp A has trust badges above fold | No badges visible | Adding trust badges will increase CVR by 10% |
| Comp B has 3-step checkout | You have 5 steps | Reducing steps will reduce abandonment by 15% |
| Comp C uses video testimonials | You have text only | Video testimonials will increase trust |

### Test Prioritization

1. **Quick wins** - Easy to implement, competitor-validated
2. **Strategic bets** - High impact, requires more effort
3. **Experiments** - Differentiation opportunities
4. **Table stakes** - Must-haves you're missing

### What NOT to Copy

| Avoid Copying | Why |
|---------------|-----|
| Brand voice | Authenticity matters |
| Exact design | Differentiation is value |
| Features you can't support | Sets wrong expectations |
| Strategies without context | They may have different goals |

## Key Takeaways

1. **Don't optimize blind**: Competitor insights reveal validated patterns
2. **Focus on CRO elements**: Homepage, landing pages, funnels, checkout
3. **Use frameworks**: ICE/PIE for prioritizing competitor-inspired tests
4. **Monitor continuously**: Markets evolve; competitors change
5. **Differentiate strategically**: Copy patterns, not brand identity
