# Webinar / Event Landing Page

> **Purpose**: Drive registrations for live or on-demand webinars and virtual events
> **Conversion Goal**: Form submission (registration)
> **SEO Priority**: Low-Medium (see notes below)
> **Framework**: PASTOR or 4Ps

---

## When to Use

| Scenario | Recommendation |
|----------|----------------|
| **Live webinar with deadline** | Urgency-focused, countdown |
| **On-demand/evergreen webinar** | Benefit-focused, always available |
| **Multi-speaker event** | Speaker credibility emphasized |
| **Product demo webinar** | Solution-focused |

---

## Key Elements

### Required Elements

| Element | Purpose |
|---------|---------|
| **Clear headline** | State the transformation or key benefit |
| **Date/Time** | When it happens (with timezone) |
| **Duration** | Set expectations (45-60 min typical) |
| **What you'll learn** | 3-5 specific takeaways |
| **Speaker credibility** | Why they're qualified |
| **Registration form** | Capture lead info |
| **Privacy reassurance** | Build trust |

### Optional Elements

| Element | When to Use |
|---------|-------------|
| **Countdown timer** | Live events with deadline |
| **Urgency messaging** | Limited seats/time-sensitive |
| **Past attendee testimonials** | Recurring webinars |
| **Company logos** | B2B credibility |
| **Agenda/outline** | Longer or complex topics |
| **Certificate mention** | Educational/professional development |

---

## Form Best Practices

### Minimum Fields

- Email (required)
- First name (required)

### Additional Fields (use sparingly)

| Field | Include When |
|-------|--------------|
| Last name | B2B, personalized follow-up |
| Company | B2B lead qualification |
| Job title | B2B segmentation |
| Phone | High-touch sales follow-up |
| Company size | Lead scoring |

> **Source**: [HubSpot - Form Optimization](https://blog.hubspot.com/marketing/optimize-landing-page-forms)

**Rule**: Every additional field reduces conversion by ~10%. Only ask for what you'll use.

## On-Demand Variations

For evergreen/on-demand webinars:

| Change | Rationale |
|--------|-----------|
| Remove date/time | Always available |
| Remove countdown | No urgency |
| Change CTA to "Watch Now" | Immediate gratification |
| Keep duration visible | Set expectations |
| Add "Instant access" messaging | Reinforce availability |

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| Too many form fields | Reduce to essentials |
| Vague headline | Be specific about the result |
| No speaker credibility | Add bio and credentials |
| No clear date/time | Make it prominent with timezone |
| Missing urgency for live events | Add countdown and seat limits |
| No mobile optimization | Test on multiple devices |
