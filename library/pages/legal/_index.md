> **Pages**: Privacy Policy, Terms of Service, Cookie Policy, Accessibility Statement
> **Principle**: Legal pages can enhance trust when done well

## Essential Legal Pages

### 1. Privacy Policy

**Purpose**: Explain how you collect, use, and protect personal data

#### Required Sections

| Section | Content |
|---------|---------|
| **Data collected** | What information you gather |
| **How it's used** | Purpose of data collection |
| **Sharing** | Third parties receiving data |
| **Storage & security** | How data is protected |
| **User rights** | Access, deletion, correction |
| **Contact** | How to reach data protection officer |

#### By Regulation

| Regulation | Additional Requirements |
|------------|------------------------|
| **GDPR (EU)** | Legal basis, data transfers, DPO contact, 72hr breach notification |
| **CCPA (California)** | "Do Not Sell" link, categories of data, financial incentives |
| **LGPD (Brazil)** | Similar to GDPR, Portuguese required |
| **PIPEDA (Canada)** | Consent requirements, access rights |

### 2. Terms of Service

**Purpose**: Define the legal agreement between you and users

#### Essential Sections

| Section | Content |
|---------|---------|
| **Acceptance** | How users agree to terms |
| **Services** | What you provide |
| **User obligations** | Expected behavior |
| **Intellectual property** | Who owns what |
| **Limitations** | Liability limits, warranties |
| **Termination** | When/how accounts end |
| **Disputes** | How conflicts are resolved |
| **Governing law** | Which jurisdiction applies |

#### Key Clauses by Business Type

| Business | Key Clauses |
|----------|-------------|
| **SaaS** | Subscription terms, SLA, data ownership |
| **E-commerce** | Returns, shipping, payment terms |
| **Marketplace** | User-generated content, disputes |
| **Content** | Licensing, attribution |

### 3. Cookie Policy

**Purpose**: Explain cookie usage and user choices

#### Cookie Categories

| Category | Purpose | Consent Required |
|----------|---------|------------------|
| **Strictly Necessary** | Site functionality | No |
| **Performance** | Analytics, monitoring | Yes (GDPR) |
| **Functional** | Preferences, features | Yes (GDPR) |
| **Marketing** | Ads, retargeting | Yes |

#### Cookie Banner Requirements

| Regulation | Requirements |
|------------|--------------|
| **GDPR** | Opt-in consent, easy reject, granular choices |
| **CCPA** | Notice, opt-out option |
| **ePrivacy** | Clear information, genuine choice |

## Cookie Consent Implementation

### Banner Best Practices

| Do | Don't |
|----|-------|
| Equal prominence for Accept/Reject | Hide reject in settings |
| Remember preferences | Ask every visit |
| Easy to change later | Make it hard to modify |
| Load cookies only after consent | Pre-load marketing cookies |
| Granular category options | All-or-nothing |

| Terms of Service | Optional | Rarely searched |

## Compliance Checklist

### Privacy Policy

- [ ] All data collection described
- [ ] Legal basis stated (GDPR)
- [ ] Third-party sharing disclosed
- [ ] User rights explained
- [ ] Contact information provided
- [ ] Last updated date visible
- [ ] Easy to find (footer link)

### Terms of Service

- [ ] Acceptance mechanism clear
- [ ] Services defined
- [ ] User responsibilities stated
- [ ] Limitation of liability included
- [ ] Dispute resolution process
- [ ] Governing law specified
- [ ] Age restrictions (if applicable)

### Cookie Policy

- [ ] All cookies listed
- [ ] Categories explained
- [ ] Third-party cookies disclosed
- [ ] Opt-out instructions provided
- [ ] Banner implements consent properly
- [ ] Preferences can be changed

### Accessibility Statement

- [ ] Conformance level stated
- [ ] Known issues documented
- [ ] Feedback mechanism provided
- [ ] Contact information included
- [ ] Remediation timeline (if issues)

---

## Key Takeaways

1. **Trust, not just compliance**: Well-designed legal pages build trust
2. **Make it readable**: Plain English summaries help users
3. **Keep it current**: Update dates build confidence
4. **Easy to find**: Footer links are standard
5. **Consent done right**: Cookie banners affect bounce rate
