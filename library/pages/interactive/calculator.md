## Why Calculators Convert

Calculators excel at the bottom of the funnel because they:

1. **Provide immediate value**: Users get personalized results
2. **Demonstrate ROI**: Justify purchase decisions with data
3. **Qualify leads**: High-intent users self-identify
4. **Build trust**: Show expertise through methodology

> "Calculators have slightly lower average conversion rates (typically 20-30%), but the leads are often high-intent and closer to purchase."
>
> *Source: [Outgrow](https://outgrow.co/blog/quiz-vs-calculator-lead-generation/)*

## Input Design

### Input Types

| Input Type | Best For | UX Considerations |
|------------|----------|-------------------|
| **Number field** | Precise values | Add +/- buttons, format thousands |
| **Slider** | Ranges, exploration | Show value, use realistic range |
| **Dropdown** | Limited options | Use clear labels, default value |
| **Radio buttons** | Binary/few choices | Horizontal on desktop |
| **Checkbox** | Multi-select features | Group logically |
| **Toggle** | Yes/no | Clear on/off states |

## Lead Capture Strategy

### When to Gate

| Strategy | Pros | Cons |
|----------|------|------|
| **Results preview + gate full report** | High capture, value shown | Some drop-off |
| **Gate before any results** | Maximum captures | Lower completion |
| **Optional gate (save/email results)** | Best UX | Lower capture rate |
| **Soft gate (get PDF/detailed breakdown)** | Good balance | Requires valuable upgrade |

## A/B Testing Ideas

### High-Impact Tests

| Element | Test Variations |
|---------|-----------------|
| **Headline** | Outcome-focused vs process-focused |
| **Input count** | Minimal (5) vs comprehensive (15) |
| **Results display** | Single metric vs full breakdown |
| **Gate timing** | Before results vs after preview |
| **CTA copy** | "Get Report" vs "See Full Analysis" |
| **Social proof** | With testimonial vs without |

### Testing Priorities

1. Gate strategy (biggest impact on leads)
2. Headline/value proposition
3. Number of inputs (completion rate)
4. Results presentation
5. CTA copy and design
