# Glossary Page Blueprint

> **SEO benefit**: Rank for "[term] definition" queries + internal linking hub
> **Best for**: Industry authority, SEO, content marketing, support
> **Link strategy**: Central hub for internal link distribution

---

## Why Glossary Pages Work

### SEO Power

> "Adding a glossary section to your company's website is a powerful way to rank for more industry terms. It will attract more backlinks organically and supercharge internal linking."
>
> *Source: [Upgrow](https://www.upgrow.io/blog/glossary-for-seo)*

### Search Intent Match

> "If the top results for your target keyword are informational ('what is', 'defined', 'beginner's guide'), you must follow suit. Look at results for a broad search like 'blockchain'â€”most top results are glossary-type pages."
>
> *Source: [Upgrow](https://www.upgrow.io/blog/glossary-for-seo)*

### Internal Linking Benefits

> "Glossary pages are natural places to point internal traffic to when defining terms. They can then pass their link juice on by linking out to more transactional pages like your product or service pages."
>
> *Source: [Upgrow](https://www.upgrow.io/blog/glossary-for-seo)*

### Linking Best Practices

> "Aim for key pages to be accessible within three clicks from the homepage. A good rule of thumb is 1-2 internal links per 300 words."
>
> *Source: [Clearscope](https://www.clearscope.io/blog/internal-links)*

| Practice | Implementation |
|----------|----------------|
| Contextual linking | Link terms naturally within content |
| First mention priority | Link the first instance, not every mention |
| Descriptive anchors | Use term name, not "click here" |
| Relevance | Only link genuinely related content |
| Reasonable density | 3-8 internal links per 1000 words |

## Content Guidelines

### Definition Quality Checklist

- [ ] Answers "What is X?" in first sentence
- [ ] Written for the target audience's level
- [ ] Avoids circular definitions
- [ ] Includes concrete examples
- [ ] Mentions why it matters
- [ ] Links to deeper resources

### Definition Format

**Basic structure**:
```
[Term] is [category/type] that [key function/characteristic].

Example: "A conversion rate is a metric that measures the percentage of
visitors who complete a desired action on your website."
```

**Expanded structure**:
```
Paragraph 1: Quick definition (featured snippet target)
Paragraph 2: Why it matters / context
Paragraph 3: How it works / process
Paragraph 4: Examples
Paragraph 5: Common misconceptions / vs other terms
```

### Term Selection Criteria

| Priority | Selection Criteria |
|----------|---------------------|
| High | Core industry concepts customers search |
| High | Terms used in your product/service |
| Medium | Technical terms in your content |
| Medium | Terms competitors rank for |
| Low | Obscure terms (unless niche authority goal) |

## Glossary Types by Industry

### SaaS/Technology

```
Focus: Technical concepts, features, metrics
Examples: API, SaaS, ARR, Churn Rate, NPS
Structure: Definition + technical explanation + use case
```

### Finance/Legal

```
Focus: Regulatory terms, financial concepts
Examples: Amortization, Fiduciary, ROI, Escrow
Structure: Definition + legal context + practical example
Note: Disclaimer about professional advice
```

### Marketing/E-commerce

```
Focus: Strategies, metrics, platforms
Examples: CTR, Retargeting, Attribution, AOV
Structure: Definition + calculation (if metric) + optimization tips
```

### Healthcare

```
Focus: Medical terms, procedures, conditions
Examples: Telehealth, Prior Authorization, HSA
Structure: Definition + patient-friendly explanation + relevance
Note: Medical disclaimer
```

## Maintenance Strategy

### Regular Updates

> "Internal linking is not a one-time job. Add more links to pillar and child pages as you expand topic clusters. Update links on old articles to point to new articles."
>
> *Source: [SurferSEO](https://surferseo.com/blog/seo-internal-linking/)*

| Task | Frequency |
|------|-----------|
| Review definitions for accuracy | Quarterly |
| Add new terms | Monthly |
| Update internal links | When new content publishes |
| Check for broken links | Monthly |
| Review analytics for gaps | Quarterly |
| Update for industry changes | As needed |

| Metric | What It Tells You |
