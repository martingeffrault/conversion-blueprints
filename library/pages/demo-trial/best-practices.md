## Form Optimization

### Field Count Impact

> "Each form field you remove can increase conversions by up to 10%."
>
> *Source: [Unbounce](https://unbounce.com/conversion-rate-optimization/b2b-conversion-rates/)*

### Demo Form Fields

| Field | Required | Why |
|-------|----------|-----|
| Work email | Yes | Qualification + contact |
| Company | Yes | Research before call |
| Team size | Depends | Routing + personalization |
| Role | No | Can gather on call |
| Phone | No | Can gather on call |
| Use case | No | Can gather on call |

**Recommended minimum: 2-3 fields**

### Trial Form Fields

| Approach | Fields | Conversion |
|----------|--------|------------|
| Email only | 1 | Highest |
| Email + name | 2 | High |
| Email + name + company | 3 | Medium |
| Full form | 5+ | Lower |

**Best practice: Email-only with progressive profiling**

### Progressive Profiling

Collect additional info after signup:
1. Email only at signup (maximum conversion)
2. Name + role during onboarding
3. Company info when value demonstrated
4. Qualifying questions before demo

## Demo vs. Trial: When to Use

### Offer Demo When

- Complex product (steep learning curve)
- Enterprise sales (high ACV)
- Multiple stakeholders
- Custom implementation required
- Pricing requires discussion

### Offer Trial When

- Self-serve product
- Quick time-to-value
- SMB/PLG motion
- Clear product-market fit
- Standard pricing

### Offer Both When

- Multiple segments
- Enterprise + SMB
- Different use cases
- Various buyer preferences

## Checklist

### Demo Page

- [ ] Value-focused headline
- [ ] Minimal form (2-3 fields)
- [ ] Instant calendar booking
- [ ] Social proof (logos + testimonial)
- [ ] Demo agenda / expectations
- [ ] Alternative options
- [ ] Trust badges
- [ ] Mobile optimized
- [ ] < 3 second load time

### Trial Page

- [ ] Outcome-focused headline
- [ ] Single-field signup (email)
- [ ] SSO options
- [ ] Trial details (duration, features)
- [ ] Product preview
- [ ] Social proof
- [ ] FAQ section
- [ ] Clear next steps post-signup
- [ ] Mobile optimized
