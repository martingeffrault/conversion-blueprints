# Homepage Blueprints

> **Purpose**: Navigate the homepage blueprints library
> **Available**: SaaS, E-commerce, Agency, Local Business, Affiliate
> **Principle**: Choose the blueprint that matches your business model

## Which Homepage Blueprint?

### Quick Decision Guide

| Your Business | Use Blueprint | Key Focus |
|---------------|---------------|-----------|

## Available Blueprints

### [SaaS Homepage](saas.md)

For B2B and B2C software products with subscription models.

**Sections covered:**
- Hero with product visualization
- Social proof (logos, metrics)
- Feature showcase
- How it works
- Pricing preview
- Testimonials
- Final CTA

**Best for:** Software companies, tools, platforms, apps

---

### [E-commerce Homepage](ecommerce.md)

**Sections covered:**
- Hero with promotions
- Category navigation
- Featured products
- Sale/seasonal sections
- Trust badges
- Newsletter capture

**Best for:** Online retail, marketplaces, DTC brands

## Common Elements Across All

### Must-Have Elements

| Element | Purpose |
|---------|---------|
| Clear value proposition | Communicate what you do in seconds |
| Primary CTA | Single most important action |
| Trust signals | Build credibility above fold |
| Navigation | Help users find what they need |
| Mobile optimization | 60%+ traffic is mobile |

### Section Order Principles

```
HOMEPAGE FLOW

1. Hero
   └── Value prop + Primary CTA

2. Trust/Credibility
   └── Logos, metrics, or reviews

3. Core Offering
   └── Products, services, or content

4. Differentiation
   └── Why choose you

5. Social Proof
   └── Testimonials, case studies

6. Secondary Content
   └── Additional features, resources

7. Final CTA
   └── Reinforce primary action
```

## Customization Guide

### When to Combine Approaches

| Scenario | Recommendation |
|----------|----------------|
| SaaS with e-commerce | Start with SaaS, add product showcase |
| Local agency | Combine local + agency patterns |
| E-commerce with subscriptions | Add SaaS elements to e-commerce |
| Affiliate with products | Blend affiliate + e-commerce |

### Testing Priorities

| Element | Impact Potential |
|---------|------------------|
| Hero headline | High |
| Primary CTA | High |
| Trust signals above fold | High |
| Section order | Medium |
| Visual design | Medium |
