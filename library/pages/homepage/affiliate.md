# Affiliate/Review Site Homepage Blueprint

> **Business Type**: Content site monetized through affiliate links, ads, or lead gen
> **Primary Goals**: Establish authority, drive organic traffic, maximize RPV
> **SEO Priority**: Critical (organic is primary traffic source)
> **Framework**: Authority + Navigation Clarity

---

## Key Differences vs Other Homepages

| Aspect | Affiliate | E-commerce | SaaS |
|--------|-----------|------------|------|
| Revenue | Affiliate commissions | Product sales | Subscriptions |
| Traffic source | Organic search | Multi-channel | Paid + organic |
| Homepage role | Hub/Navigation | Storefront | Conversion |
| Content depth | Very deep | Product-focused | Feature-focused |

## E-E-A-T for Affiliate Sites

### Experience

| Element | Implementation |
|---------|----------------|
| Hands-on testing | "We tested this for X weeks" |
| Real photos | Non-stock, taken during testing |
| Methodology | Documented testing process |
| Updates | "Last tested: [date]" |

### Expertise

| Element | Implementation |
|---------|----------------|
| Author bios | Expert backgrounds |
| Credentials | Relevant qualifications |
| Industry experience | Years in field |
| Specializations | Specific expertise areas |

### Authoritativeness

| Element | Implementation |
|---------|----------------|
| Press mentions | "As seen in" section |
| Awards | Industry recognition |
| Citations | Being referenced by others |
| Backlinks | Quality external links |

### Trustworthiness

| Element | Implementation |
|---------|----------------|
| Affiliate disclosure | Clear, above the fold |
| Review methodology | Transparent process |
| No fake reviews | Real testing only |
| Update dates | Show freshness |

## Content Types to Feature

| Content Type | Homepage Prominence | Purpose |
|--------------|---------------------|---------|
| Best [X] lists | High | Money pages, high intent |
| Product reviews | Medium | Individual product coverage |
| Comparisons | Medium | Decision-stage content |
| Buying guides | Medium | Educational, top-funnel |
| How-to articles | Low | Informational, traffic drivers |
| News/updates | Low | Freshness signals |

## Schema Markup

### Organization

```json
{
  "@type": "Organization",
  "name": "Site Name",
  "url": "https://example.com",
  "logo": "https://example.com/logo.png",
  "sameAs": [
    "https://twitter.com/sitename",
    "https://youtube.com/sitename"
  ]
}
```

### WebSite with Search

```json
{
  "@type": "WebSite",
  "url": "https://example.com",
  "potentialAction": {
    "@type": "SearchAction",
    "target": "https://example.com/search?q={search_term}",
    "query-input": "required name=search_term"
  }
}
```

### ItemList for Featured Content

```json
{
  "@type": "ItemList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "url": "https://example.com/best-product-guide/"
    }
  ]
}
```

## Performance Metrics

| Metric | Target | Why |
|--------|--------|-----|
| Time on site | > 3 min | Content engagement |
| Pages per session | > 2 | Internal linking |
| Bounce rate | < 50% | Content relevance |
| Click-through to articles | > 30% | Homepage effectiveness |
| Affiliate click rate | Varies | Revenue optimization |

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| No clear navigation | Category structure visible |
| Missing affiliate disclosure | Add prominently |
| No author/expertise signals | Add bios, credentials |
| Thin homepage content | Add value with curated content |
| No update dates | Show freshness |
| Too many ads above fold | Content-first approach |
| No E-E-A-T signals | About page, methodology page |
