# Agency Homepage Blueprint

> **Business Type**: Service-based agency (design, marketing, development, consulting)
> **Primary Goals**: Generate qualified leads, showcase expertise, build trust
> **SEO Priority**: High (brand + service keywords)
> **Framework**: StoryBrand + Expertise Demonstration

## SEO Considerations

### Target Keywords

| Page Element | Keyword Target |
|--------------|----------------|
| Title tag | "[Service] agency in [Location]" |
| H1 | Value proposition (can include keywords) |
| Services section | "[Specific service]" |
| Meta description | Compelling + location + service |

### Local SEO (if applicable)

- Include location in meta title
- Add Google Business Profile link
- Schema markup: LocalBusiness or Organization
- NAP consistency (Name, Address, Phone)

### Service-Specific Pages

Create dedicated pages for each service targeting:
- "[service] agency"
- "[service] services"
- "[service] company"
- "[service] for [industry]"

## Trust Elements Priority

| Priority | Element | Purpose |
|----------|---------|---------|
| 1 | Client logos | Instant credibility |
| 2 | Case studies with results | Proof of capability |
| 3 | Specific metrics | Quantified success |
| 4 | Named testimonials | Personal validation |
| 5 | Awards/certifications | Third-party validation |

### Primary CTA

- "Get a Free Consultation"
- "Request a Quote"
- "Let's Talk"
- "Start Your Project"

### Secondary CTAs

- "See Our Work"
- "View Case Studies"
- "Download Our Portfolio"

## Agency-Specific Variations

### Design Agency

- Visual-heavy portfolio
- Larger project images
- Animation/motion if appropriate
- Minimal text, maximum visual impact

### Marketing Agency

- Results-focused (metrics prominently displayed)
- Case studies with ROI
- Strategy-focused process section
- Blog/insights prominence

### Development Agency

- Technical credibility
- Tech stack showcase
- GitHub/code examples
- Security/performance metrics

### Consulting Agency

- Thought leadership emphasis
- Principal/partner bios
- Publications/speaking
- Industry expertise

---

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| Generic "We are a creative agency" | Specific value prop with target audience |
| No clear services | Clear service categories |
| Portfolio without context | Add challenge/solution/results |
| Missing results/metrics | Add specific outcomes |
| No social proof | Add logos and testimonials |
| Weak CTA | Strong, specific action |
