# Contact Page Blueprint

> **Purpose**: Enable visitors to reach out, start conversations, generate leads
> **Audience**: Prospects, customers, partners, media
> **Goal**: Maximize form submissions, reduce friction, provide multiple contact options

---

## Core Principle

The contact page is a conversion page. Every element should reduce friction and encourage visitors to reach out. Make it easy, not bureaucratic.

## Contact Form Design

### Essential Fields

| Field | When Required | Notes |
|-------|---------------|-------|
| **Name** | Always | First name or full name |
| **Email** | Always | For response |
| **Message** | Always | Free-form inquiry |
| **Phone** | Sometimes | If calls are preferred |
| **Company** | B2B | Helps qualify leads |
| **Subject/Type** | High volume | Route inquiries |

### Optional Fields (Use Sparingly)

| Field | Use When |
|-------|----------|
| Budget | Sales inquiries, agencies |
| Timeline | Project inquiries |
| How did you hear about us? | Marketing attribution |
| Job title | Enterprise sales |
| Company size | B2B qualification |

### Form Field Best Practices

**Do**:
- Mark required fields clearly (*)
- Use placeholder text wisely
- Provide helpful error messages
- Auto-format phone numbers
- Use appropriate input types (email, tel)

**Don't**:
- Require unnecessary fields
- Use CAPTCHA unless spam is severe (try honeypot first)
- Make message field tiny
- Reset form on validation error

## Common Mistakes

### âŒ Form Too Long

More fields = fewer submissions. Only ask what you need.

### âŒ No Alternative Contact

Some people won't fill forms. Give them options.

### âŒ Hidden Contact Info

Don't make people hunt for phone/email.

### âŒ Generic Headline

"Contact Us" is boring. Try "Let's Talk" or "How Can We Help?"

### âŒ No Response Expectation

People wonder if their message went into a void.

### âŒ No Confirmation

Always confirm form submission clearly.
