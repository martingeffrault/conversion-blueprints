# Pre-Launch Checklist

> **Purpose**: Ensure nothing is forgotten before going live
> **Impact**: Prevents embarrassing mistakes, ensures smooth launch
> **Usage**: Complete every item before launch; don't skip any section

---

## Content Checklist

### Text Content

- [ ] All placeholder text (Lorem ipsum) removed
- [ ] Spelling and grammar checked (use Grammarly, Hemingway)
- [ ] All headlines reviewed for clarity and impact
- [ ] Copyright dates current (© 2024)
- [ ] Company name consistent throughout
- [ ] Contact information accurate
- [ ] Phone numbers tested (clickable on mobile)
- [ ] Email addresses working and tested
- [ ] Address/location information correct
- [ ] No broken internal links
- [ ] External links open in new tabs
- [ ] Dead links checked (use Screaming Frog, ahrefs)

### Legal Content

- [ ] Privacy Policy complete and accessible
- [ ] Terms of Service/Terms of Use complete
- [ ] Cookie Policy (if using cookies)
- [ ] Affiliate disclosure (if applicable)
- [ ] DMCA notice (if user content)
- [ ] Refund policy (if e-commerce)
- [ ] Shipping policy (if physical goods)
- [ ] Accessibility statement

### Media Content

- [ ] All placeholder images replaced
- [ ] Image alt text on all images
- [ ] Images optimized (WebP, compressed)
- [ ] Logos in proper resolution
- [ ] Videos hosted properly (YouTube/Vimeo or CDN)
- [ ] Video thumbnails set
- [ ] Downloadable files tested
- [ ] PDF links working

## Design & UX Checklist

### Visual Design

- [ ] Consistent typography throughout
- [ ] Consistent color usage (matches brand)
- [ ] Consistent spacing/padding
- [ ] Icons consistent in style
- [ ] Buttons consistent in style
- [ ] Forms visually consistent
- [ ] No design inconsistencies between pages
- [ ] Favicon set (all sizes)
- [ ] Apple touch icons set
- [ ] Open Graph images for social sharing

### User Experience

- [ ] Navigation clear and intuitive
- [ ] Important pages accessible in 3 clicks
- [ ] 404 page designed and helpful
- [ ] Error messages user-friendly
- [ ] Success messages confirm actions
- [ ] Loading states for slow operations
- [ ] Back button works correctly
- [ ] Scroll behavior smooth
- [ ] Focus states visible (accessibility)
- [ ] Skip navigation link present

### Responsive Design

- [ ] Desktop (1920px) tested
- [ ] Laptop (1440px, 1366px) tested
- [ ] Tablet landscape (1024px) tested
- [ ] Tablet portrait (768px) tested
- [ ] Mobile (375px, 414px) tested
- [ ] Mobile landscape tested
- [ ] Touch targets minimum 44px
- [ ] No horizontal scrolling on mobile
- [ ] Text readable without zooming
- [ ] Images scale properly

## Technical Checklist

### Performance

- [ ] Page speed tested (Google PageSpeed Insights)
- [ ] Core Web Vitals passing
  - [ ] LCP < 2.5s
  - [ ] FID < 100ms
  - [ ] CLS < 0.1
- [ ] Images lazy-loaded
- [ ] JavaScript minified
- [ ] CSS minified
- [ ] Gzip/Brotli compression enabled
- [ ] Browser caching configured
- [ ] CDN configured (if applicable)
- [ ] No render-blocking resources
- [ ] Critical CSS inlined

### Security

- [ ] SSL certificate installed
- [ ] HTTPS redirects working
- [ ] SSL certificate auto-renewing
- [ ] Security headers configured
  - [ ] X-Frame-Options
  - [ ] X-Content-Type-Options
  - [ ] Content-Security-Policy
  - [ ] Strict-Transport-Security
- [ ] Admin URLs protected
- [ ] Sensitive data encrypted
- [ ] Input sanitization active
- [ ] SQL injection prevention
- [ ] XSS prevention
- [ ] Rate limiting on forms

### Server & Hosting

- [ ] Production server configured
- [ ] Domain DNS pointed correctly
- [ ] WWW vs non-WWW redirects
- [ ] Staging environment cleaned up
- [ ] Debug mode disabled
- [ ] Error logging configured
- [ ] Backup system active
- [ ] Monitoring alerts configured
- [ ] Server timezone correct

## SEO Checklist

### On-Page SEO

- [ ] Every page has unique title tag
- [ ] Every page has meta description
- [ ] H1 tag on every page (only one)
- [ ] Heading hierarchy correct (H1→H2→H3)
- [ ] URLs are clean and descriptive
- [ ] Internal linking structure solid
- [ ] Images have alt text
- [ ] No duplicate content issues
- [ ] Canonical tags set correctly

### Technical SEO

- [ ] XML sitemap generated
- [ ] XML sitemap submitted to Search Console
- [ ] robots.txt configured
- [ ] robots.txt not blocking important pages
- [ ] 301 redirects from old URLs (if migration)
- [ ] No broken links (404s)
- [ ] Schema markup implemented
- [ ] Mobile-friendly test passing
- [ ] Site indexable (no noindex on important pages)

### Local SEO (if applicable)

- [ ] Google Business Profile claimed
- [ ] NAP (Name, Address, Phone) consistent
- [ ] Local schema markup
- [ ] Location pages created
- [ ] Google Maps embedded

## Pre-Launch Final Steps

### One Week Before

- [ ] Final content review complete
- [ ] All stakeholders approved
- [ ] Launch announcement prepared
- [ ] Support team briefed
- [ ] Social media posts scheduled
- [ ] Email announcement ready

### Day Before

- [ ] Full site backup created
- [ ] Rollback plan documented
- [ ] Team on standby
- [ ] Test all critical paths one more time
- [ ] Domain propagation check (if new domain)

### Launch Day

- [ ] DNS changes propagated
- [ ] SSL working on production
- [ ] Analytics receiving data
- [ ] Forms tested on production
- [ ] Payment processing live
- [ ] 301 redirects active
- [ ] Old site redirecting (if migration)
