# Launch Checklists

> **Purpose**: Systematic approach to launching websites without missing critical steps
> **Impact**: Prevents embarrassing mistakes, ensures professional launches
> **Usage**: Work through checklists sequentially, don't skip sections

## Quick Reference

### Must-Have Before Launch

1. SSL certificate installed
2. All content finalized
3. Forms tested and working
4. Analytics tracking verified
5. Legal pages complete
6. Backup system active
7. Redirects in place (if migration)
8. Stakeholder approval

### Critical Day-1 Checks

1. Site accessible to all users
2. Analytics receiving data
3. Forms submitting correctly
4. Payments processing (if applicable)
5. No critical errors in console
