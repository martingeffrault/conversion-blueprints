## Team Card Elements

### Essential Elements

| Element | Purpose |
|---------|---------|
| Photo | Humanization, recognition |
| Name | Identification |
| Title/Role | Expertise signaling |

### Optional Elements

| Element | When to Include |
|---------|-----------------|
| Bio (short) | Leadership, key roles |
| Bio (full) | Dedicated team pages |
| Social links | Personal brands, thought leaders |
| Email | Direct contact appropriate |
| Location | Distributed teams |
| Fun fact | Culture emphasis |
| Skills/Tags | Technical roles |

---

## Photo Guidelines

### Technical Requirements

| Aspect | Recommendation |
|--------|----------------|
| Aspect ratio | 1:1 (square) or 4:5 |
| Minimum size | 400x400px |
| File format | JPEG (optimized) or WebP |
| Background | Consistent across team |
| Lighting | Professional, even |

### Style Consistency

| Element | Keep Consistent |
|---------|-----------------|
| Background | Same color/style |
| Framing | Same crop (shoulders up) |
| Expression | Aligned with brand |
| Attire | Appropriate for industry |
| Editing | Same treatment |

---

## Bio Writing

### Short Bio (50-100 words)

```
[Name] is [Title] at [Company]. With [X] years in [industry/field],
[he/she/they] [key accomplishment or expertise area].

Previously at [notable company] and [credentials].

[Personal touch: passion, hobby, location].
```

### Credential Emphasis

For expertise-driven businesses:

```
[Name], [Credentials]
[Title] | [Company]

[Degree] from [University]
[X]+ years in [specialty]
Featured in [Publications]
```

## Social Links

### Which to Include

| Platform | When to Include |
|----------|-----------------|
| LinkedIn | Almost always (professional) |
| Twitter/X | Thought leaders, tech |
| GitHub | Developers, open source |
| Dribbble | Designers |
| Personal site | Consultants, freelancers |
| Email | Direct contact appropriate |

### Icon Style

- Match brand aesthetic
- Consistent size
- Subtle (not distracting)
- Accessible (aria labels)

## Accessibility

| Requirement | Implementation |
|-------------|----------------|
| Alt text | Descriptive photo alt |
| Focus states | Visible on cards/links |
| Screen reader | Semantic headings |
| Color contrast | Text readable |
| Link context | "Jane Smith's LinkedIn profile" |

## Industry Examples

### Law Firm

- Emphasize credentials (JD, bar admissions)
- Professional, formal photos
- Practice areas per attorney
- Notable cases/results

### Tech Startup

- Casual but professional photos
- Previous companies (logos)
- Technical specialties
- Fun facts/personality

### Design Agency

- Creative/artistic photos
- Portfolio links
- Design tools/specialties
- Award mentions

### Healthcare

- Credentials prominent (MD, RN, etc.)
- Professional headshots
- Specializations
- Hospital affiliations

---

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| Inconsistent photos | Standardize style |
| Missing photos | Invest in professional shots |
| Generic stock photos | Use real team photos |
| Too many team members | Curate for relevance |
| No hierarchy | Lead with leadership |
| Missing credentials | Add for trust-building |
| Dead social links | Audit regularly |
