### Card Zones

| Zone | Purpose | Content |
|------|---------|---------|
| **Visual** | Attract attention | Image, icon, illustration |
| **Eyebrow** | Categorize/label | Tag, badge, category |
| **Title** | Identify content | Heading (required) |
| **Body** | Explain content | Description, details |
| **Meta** | Context | Date, author, stats |
| **Actions** | Enable interaction | Buttons, links |

## Card Sizing & Spacing

### Image Ratios

| Ratio | Use Case | Visual |
|-------|----------|--------|
| 1:1 (Square) | Products, profiles | Compact, uniform |
| 4:3 | Blog posts, general | Standard, familiar |
| 16:9 | Video thumbnails | Widescreen |
| 3:2 | Photography | Natural photos |
| 2:3 (Portrait) | People, books | Vertical focus |

### Card Dimensions

| Context | Min Width | Max Width | Notes |
|---------|-----------|-----------|-------|
| Grid (3 col) | 280px | 400px | Responsive between |
| Grid (4 col) | 220px | 300px | Tighter content |
| Featured | 100% | 800px | Full width |
| Sidebar | 250px | 350px | Fixed width |

### Spacing Recommendations

| Element | Spacing |
|---------|---------|
| Card padding | 16-24px |
| Between cards | 16-32px (gap) |
| Title to description | 8-12px |
| Description to meta | 16px |
| Meta to actions | 16-24px |

## Card Accessibility

### Requirements

| Requirement | Implementation |
|-------------|----------------|
| **Focus visible** | Clear focus ring on interactive cards |
| **Keyboard nav** | Tab through cards, Enter to activate |
| **Alt text** | Meaningful descriptions for images |
| **Color contrast** | 4.5:1 for text, 3:1 for UI elements |
| **Touch targets** | Minimum 44x44px for interactive areas |

### Card Link Patterns

**Single link (entire card)**:
```html
<article class="card">
  <a href="/page" class="card__link">
    <!-- Card contents wrapped in link -->
  </a>
</article>
```

**Multiple links**:
```html
<article class="card">
  <a href="/category">Category</a>
  <h3><a href="/page">Title</a></h3>
  <p>Description</p>
  <a href="/page">Read more</a>
</article>
```

---

## Card Anti-Patterns

| Anti-Pattern | Problem | Solution |
|--------------|---------|----------|
| Too much text | Overwhelming | Truncate with "..." |
| No visual hierarchy | Hard to scan | Clear title, lighter description |
| Inconsistent heights | Messy grid | Equal heights or masonry |
| Missing images | Text walls | Use icons if no images |
| Tiny click areas | Hard to interact | Expand click area |
| No hover feedback | Feels unresponsive | Add hover state |
| Uneven spacing | Looks amateur | Consistent padding/margins |
