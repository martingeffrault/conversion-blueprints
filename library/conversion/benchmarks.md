## E-Commerce Benchmarks

### Conversion Rates by Device

| Device | Average CVR | Gap |
|--------|-------------|-----|
| Desktop | 4.3% - 4.8% | Baseline |
| Mobile | 2.2% - 2.9% | ~2× lower |
| Tablet | 3.5% | Middle |

*Sources: [SQ Magazine](https://sqmagazine.co.uk/mobile-vs-desktop-statistics/), [Smart Insights](https://www.smartinsights.com/ecommerce/ecommerce-analytics/ecommerce-conversion-rates/)*

### Cart Abandonment

| Metric | Rate |
|--------|------|
| Global cart abandonment | 70.19% |
| Mobile cart abandonment | 77.2% |
| Desktop cart abandonment | 65% |
| Annual revenue lost | $18 billion |

*Source: [Baymard Institute](https://baymard.com/research/checkout-usability)*

### Top Abandonment Reasons

| Reason | Percentage |
|--------|------------|
| Extra costs (shipping, taxes, fees) | 48% |
| Required account creation | 26% |
| Too long/complicated checkout | 22% |
| Couldn't see total cost upfront | 21% |
| Didn't trust site with payment info | 18% |
| Website errors | 17% |
| Slow delivery | 16% |

*Source: [Baymard Institute](https://baymard.com/research/checkout-usability)*

### Checkout Optimization Impact

| Optimization | Impact |
|--------------|--------|

| Mobile wallet unavailable | 40% abandon cart |
| Exit popup with discount | 17.12% recovery rate |

*Sources: [BigCommerce](https://www.bigcommerce.com/articles/ecommerce/checkout-optimization/), [OptiMonk](https://www.optimonk.com/popup-statistics/)*

## Form Conversion Benchmarks

### Field Count Impact

| Fields | CVR Change | Source |
|--------|------------|--------|
| 11 → 4 fields | +160% | Imagescape |
| 9 → 5 fields | +34% | Marketo |
| 1 optional field removed | +$12M/year | Expedia |
| Streamlined lead form | +53.8% revenue/visitor | FSAstore |

*Source: [Venture Harbour](https://ventureharbour.com/how-form-length-impacts-conversion-rates/)*

### Optimal Field Count

| Form Type | Sweet Spot | Notes |
|-----------|------------|-------|
| Email signup | 1-2 fields | Email only or Email + Name |
| Lead gen | 3-4 fields | Name, Email, Company, Need |
| Contact | 4-5 fields | Name, Email, Phone, Message, Subject |
| Checkout | Minimal | Auto-fill, no unnecessary fields |

## Page Speed Impact

### Conversion by Load Time

| Load Time | Conversion Rate | Change |
|-----------|-----------------|--------|
| 1 second | 39% | Baseline |
| 2 seconds | ~32% | -18% |
| 3 seconds | ~26% | -33% |
| 4 seconds | ~18% | -54% |
| 5+ seconds | 0.6% | -98% |

*Source: [Google/Deloitte](https://hostingstep.com/core-web-vitals-stats/)*

### Speed Optimization ROI

| Improvement | Result |
|-------------|--------|
| 0.1s faster | +8.4% CVR (retail) |
| 0.1s faster | +9.2% AOV |
| LCP +31% (Vodafone) | +8% sales |
| Core Web Vitals optimized | +53% revenue/visitor (Rakuten) |
| INP -72% (redBus) | +7% sales |

*Source: [Core Web Vitals studies](https://hostingstep.com/core-web-vitals-stats/)*

### Core Web Vitals Targets

| Metric | Good | Needs Improvement | Poor |
|--------|------|-------------------|------|
| LCP (Largest Contentful Paint) | ≤2.5s | 2.5s-4s | >4s |
| INP (Interaction to Next Paint) | ≤200ms | 200ms-500ms | >500ms |
| CLS (Cumulative Layout Shift) | ≤0.1 | 0.1-0.25 | >0.25 |

### User Behavior

| Metric | Statistic |
|--------|-----------|
| Abandon if >3s load | 53% |
| Bounce increase per 1s delay | +32% |
| Page speed influences purchase | 70% |
| Good CWV vs poor CWV | 2× conversion |

| Standard popup | 3.49% | 10%+ |

*Source: [Popupsmart Benchmark Report](https://popupsmart.com/blog/popup-conversion-benchmark-report)*

### Field Count Impact (Popups)

| Fields | CVR |
|--------|-----|
| 1 field (email only) | 5.77% |
| 2 fields | ~5% |
| 4 fields | Up to 9% (strong offer) |
| 6+ fields | Declining |

### Design Elements Impact

| Element | Impact |
|---------|--------|
| With image vs without | +63.5% CVR |
| Bottom-center position | 12.88% CVR (highest) |
| With lead magnet (mobile) | +102% CVR |
| With lead magnet (desktop) | +155% CVR |
| Time-on-page trigger | +25% vs random |

### Timing Best Practices

| Trigger | Performance |
|---------|-------------|
| 25 seconds + 55% scroll | 7% CVR |
| Exit intent | 4-5% CVR |
| Immediate (on load) | Lower, annoying |
| After engagement | Best |
