## Headline Formulas

### Formula 1: Value Proposition

> "[Achieve outcome] [without pain point]"

Examples:
- "Build websites without code"
- "Track time without the hassle"
- "Manage projects without spreadsheets"

### Formula 2: Specific Outcome

> "[Specific result] in [timeframe]"

Examples:
- "Double your leads in 30 days"
- "Learn Spanish in 3 months"
- "Ship features 10x faster"

### Formula 3: The "Better Way"

> "The [better/easier/faster] way to [desired action]"

Examples:
- "The better way to manage money"
- "The faster way to ship code"
- "The easier way to learn design"

### Formula 4: Direct Statement

> "We [do what you need]"

Examples:
- "We make meetings shorter"
- "We help teams collaborate"
- "We simplify your finances"

### What Works Best

| Headline Type | Performance |
|---------------|-------------|
| Benefit-focused | Highest |
| Specific numbers | High |
| Question format | Medium |
| Clever/punny | Low |
| Generic/vague | Lowest |

**Rule**: Clarity beats cleverness every time.

## Visual Approaches

### Product Screenshots

**Best practices**:
- Show real UI (not mockups)
- Highlight key feature
- Use realistic data
- Avoid generic content
- Show outcome, not process

**Do**: Show a completed dashboard with data
**Don't**: Show empty states or "lorem ipsum"

### Video

| Type | Impact | Best For |
|------|--------|----------|
| Auto-play (muted) | +86% CVR | Product demos |
| Click-to-play | Lower engagement | Testimonials |
| Background loop | Ambiance | Lifestyle brands |

**Video statistics**:
- Explainer video on landing page: up to +80-86% CVR â€” [EyeView via Saleslion](https://saleslion.io/sales-statistics/using-video-on-landing-pages-can-increase-conversion-by-80/)
- Video CTA: +380% clicks vs sidebar CTAs â€” [Wistia via Firework](https://firework.com/blog/impact-video-content-landing-page-conversion-rates)
- Auto-play requirement: MUST be muted

### Illustrations

**When to use**:
- Abstract concepts
- Developer tools (avoid stock photos)
- Playful brands
- Complex ideas

**Trending styles (2025)**:
- 3D isometric
- Hand-drawn accents
- Gradient meshes
- Minimal line art

### Animation

| Type | Impact | Use Case |
|------|--------|----------|
| Micro-interactions | +15% engagement | CTAs, icons |
| Scroll-triggered | Good | Progressive reveal |
| Background motion | Atmospheric | Premium feel |
| Heavy animation | Can hurt | Distracting |

**Rule**: Animation should guide attention, not steal it.

### Mobile Rules

| Rule | Why |
|------|-----|
| Single column | Screen width constraint |
| Visual above/below, not beside | Stacking |
| Full-width CTA | Easy to tap |
| Shorter headline | Limited space |
| No auto-play video | Data costs |
| Fast loading (<2s) | Impatient users |

## A/B Testing Ideas

### High-Impact Hero Tests

| Element | Variations to Test |
|---------|-------------------|
| Headline | Benefit vs. feature focus |
| Headline | Short (3-5 words) vs. longer |
| Visual | Screenshot vs. illustration |
| Visual | With people vs. without |
| CTA | "Start Free" vs. "Get Started" |
| Layout | Centered vs. split |
| Trust | With vs. without |
| Video | With vs. without |

### Test Priorities

1. **Headline** â€” Highest impact, test first
2. **CTA copy** â€” Quick win
3. **Layout** â€” Major change
4. **Visual type** â€” Depends on product
5. **Trust elements** â€” Fine-tuning

## Anti-Patterns to Avoid

| Anti-Pattern | Problem | Better Approach |
|--------------|---------|-----------------|
| Slider/carousel | -30% CVR, splits focus | Single static hero |
| Stock photos | Generic, low trust | Real product/custom |
| Clever puns | Confusing | Clear value prop |
| Too many CTAs | Decision paralysis | 1 primary + 1 secondary |
| Auto-play video with sound | Annoying, accessibility | Muted or click-to-play |
| Huge headline (50px+) | Hard to read | 32-48px optimal |
| No CTA above fold | Missed conversions | Always include CTA |
| Generic subheadline | Wastes space | Specific benefit |

## Hero by Page Type

### Homepage

```
Focus: Brand + Primary action
Elements: Full value prop, main CTA, trust signals
Goal: Get to product/signup
```

### Product Page

```
Focus: Specific product/feature
Elements: Feature headline, product visual, feature CTA
Goal: Learn more / Try feature
```

### Landing Page (PPC)

```
Focus: Single conversion goal
Elements: Offer-focused headline, form or CTA, urgency
Goal: Convert visitor
```

### Blog Post

```
Focus: Content preview
Elements: Title, meta (author, date, read time), featured image
Goal: Start reading
```
