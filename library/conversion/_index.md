## Quick Reference

### The Conversion Formula

```
Conversion = Motivation × Ability × Trigger
                ↑            ↑         ↑
           Value prop    Low friction  Right timing
```

### Key Statistics

| Optimization | Typical Impact |
|--------------|----------------|
| Single CTA vs 5+ | +28% CVR |
| Personalized CTAs | +202% CVR |
| 11→4 form fields | +160% CVR |
| Exit-intent popup | 4-5% CVR |
| Social proof (5+ reviews) | +270% CVR |
| 1s faster load time | +8.4% CVR |
| Multi-step forms | +86% CVR |

### The Trust Stack

Every high-converting page includes:

1. **Above the fold**: Clear value proposition + primary CTA
2. **Social proof**: Logos, testimonials, or stats
3. **Risk reducers**: Guarantees, security badges, "no spam"
4. **Friction minimizers**: Few fields, guest checkout, clear process
