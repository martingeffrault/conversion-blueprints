## The Modern SaaS Homepage Formula

### Pattern: The "Stripe Model"

Stripe consistently ranks as the best SaaS website. Here's their formula:

```
┌─────────────────────────────────────────────────────────────────┐
│  [Logo]            [Products ▼] [Solutions ▼] [Developers]  [CTA] │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  HERO: Concise headline (3-7 words)                            │
│  Subheadline: One sentence value prop                          │
│  [Primary CTA]  [Secondary CTA]                                │
│  Optional: Product visual / animation                          │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│  SOCIAL PROOF BAR: Logo cloud ("Used by")                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  FEATURES: 3-4 key capabilities with icons                     │
│  Visual demonstrations of each feature                         │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│  USE CASES / SOLUTIONS: Industry-specific applications         │
├─────────────────────────────────────────────────────────────────┤
│  TESTIMONIALS / CASE STUDIES: Named customers, metrics         │
├─────────────────────────────────────────────────────────────────┤
│  FINAL CTA: Strong closing with risk reducers                  │
├─────────────────────────────────────────────────────────────────┤
│  FOOTER: Comprehensive navigation, legal, social               │
└─────────────────────────────────────────────────────────────────┘
```

### Key Characteristics

| Element | Stripe's Approach | Why It Works |
|---------|-------------------|--------------|
| **Hero headline** | 3-7 words max | Instant comprehension |
| **Hero visual** | Product animation | Shows, doesn't tell |
| **CTA hierarchy** | 1 primary + 1 secondary | Clear action path |
| **Social proof** | Logo bar immediate | Builds trust fast |
| **Features** | 3-4 max per section | Avoids overwhelm |
| **Spacing** | Generous whitespace | Premium feel |
| **Animation** | Subtle, purposeful | Guides attention |

## Key Takeaways

### What Top Performers Do Differently

| Practice | Average Site | Top Performer |
|----------|--------------|---------------|
| Hero headline length | 8-15 words | 3-7 words |
| CTAs per page | 5+ | 2-3 (clear hierarchy) |
| Page load time | 4+ seconds | <2 seconds |
| Social proof placement | Footer | Above fold |
| Feature count per section | 6+ | 3-4 max |
| Whitespace | Minimal | Generous |
| Animation | Decorative | Purposeful |

### The "Linear" Effect

Linear has become a template for modern SaaS because:
1. **Ultra-minimal hero**: 4 words + 1 sentence
2. **Instant credibility**: Logo cloud immediately
3. **Feature storytelling**: Each feature is a mini-narrative
4. **Dark mode default**: Premium, developer-focused feel
5. **Smooth animations**: Polished but not distracting

### The "Stripe" Effect

Stripe sets the standard for:
1. **Information architecture**: Complex product, simple presentation
2. **Progressive disclosure**: More detail as you scroll
3. **Developer-first**: Code examples in hero
4. **Trust signals**: Enterprise logos prominently
5. **Documentation quality**: Part of the marketing
